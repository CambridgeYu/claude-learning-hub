# 🚀 快速开始指南

欢迎使用 Claude Learning Hub！这是您的5分钟快速上手指南。

## 📌 第一步：打开学习中心

**双击 `index.html` 文件**，它会在您的默认浏览器中打开。

您会看到一个漂亮的汇总页面，包含：
- 📊 统计面板（14个工具、8个类别）
- 🔍 搜索框（可以快速查找工具）
- 🏷️ 分类标签（全部/核心技能/高级应用/生产环境/开发工具）
- 🚀 快速开始区（推荐新手优先学习的4个工具）

## 🎯 第二步：选择学习路径

### 如果您是初学者 👶

**推荐从"快速开始"区的4个工具开始：**

1. **💬 Prompt Engineering Lab** 
   - 学习如何写好Prompt
   - 6种模板快速上手
   - 实时分析和优化建议

2. **🔧 Function Calling Playground**
   - 让AI能够调用工具
   - 可视化工具定义
   - 理解API集成

3. **⚡ Streaming Demo**
   - 对比流式vs非流式输出
   - 直观感受性能差异
   - 理解用户体验优化

4. **🖥️ CLI Tutorial**
   - 5步渐进式教程
   - 从安装到高级用法
   - 一键复制命令

### 如果您有基础 🎓

**可以按类别浏览：**

- **核心技能** → Prompt工程、函数调用、流式输出、技能库
- **高级应用** → RAG系统、Agent编排、多模态、自动记忆
- **生产环境** → 安全测试、评估仪表板、监控、成本优化
- **开发工具** → CLI教程、MCP连接器

## 💡 第三步：开始学习

### 使用技巧

1. **点击"启动 →"按钮**
   - 在汇总页面找到感兴趣的工具
   - 点击右下角的"启动 →"按钮
   - 会自动在新标签页打开该工具

2. **使用搜索功能**
   - 在顶部的搜索框输入关键词
   - 例如："prompt"、"security"、"cost"
   - 会自动过滤显示相关工具

3. **分类筛选**
   - 点击顶部的分类标签
   - 例如点击"核心技能"只看基础工具
   - 点击"全部"恢复显示所有工具

4. **查看文件路径**
   - 每个工具卡片底部显示文件路径
   - 例如：`prompt-engineering/prompt-lab.html`
   - 方便您直接找到文件

## ❓ 常见问题

**Q: 需要安装什么软件吗？**  
A: 不需要！只需要一个现代浏览器（Chrome、Firefox、Edge、Safari等）。

**Q: 需要联网吗？**  
A: 基本功能不需要联网。但Chart.js图表库需要从CDN加载，如果完全离线，图表可能无法显示，但不影响其他功能。

**Q: 可以在手机上使用吗？**  
A: 可以！所有页面都是响应式设计，支持手机和平板。

**Q: 如何知道我的学习进度？**  
A: 建议自己制定学习计划，每完成一个工具打个勾。README.md中有详细的学习路径建议。

**Q: 这些工具是真实的AI应用吗？**  
A: 这些是交互式学习范例，大部分使用模拟数据来展示功能和UI设计。它们是学习工具，帮助您理解概念和界面设计。

**Q: 可以用于教学吗？**  
A: 当然可以！这些工具非常适合用于AI开发教学、培训演示和技术分享。
