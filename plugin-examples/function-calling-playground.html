⚡ 并行 vs 串行调用对比

并行调用（Parallel）

同时调用多个工具，提高效率

场景：需要多个独立数据源
优势：减少总耗时
示例：同时查询天气、股票、新闻

串行调用（Sequential）

按顺序调用，后一个依赖前一个结果

场景：存在依赖关系
优势：逻辑清晰，易于调试
示例：先查城市ID，再查天气
