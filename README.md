# 🎓 Claude Learning Hub - 完整学习中心

这是一个完整的Claude AI开发交互式学习工具包，包含14个HTML学习范例，涵盖从入门到高级的所有核心技术。

## 📦 使用说明

### 快速开始

1. **打开汇总页面**
   - 双击 `index.html` 文件
   - 在浏览器中查看所有学习工具的导航界面

2. **启动学习工具**
   - 在汇总页面点击任意工具的"启动 →"按钮
   - 或直接打开对应文件夹中的HTML文件

3. **分享给他人**
   - 直接压缩整个 `claude-learning-hub` 文件夹
   - 发送压缩包给对方
   - 对方解压后双击 `index.html` 即可使用

3. **全屏模式** - 按F11获得更好的学习体验
4. **开发者工具** - 按F12查看源码学习实现细节
